# 🚗 Telegram-бот для Автосервиса

Простой Telegram-бот для автосервиса, написанный на Python с использованием фреймворка **aiogram 3.x**.

## 📌 Функционал
- 🛠 **Услуги и цены** — краткий прейскурант популярных работ.
- 📅 **Записаться на ТО** — инструкция и форма подачи заявки на ремонт.
- 📍 **Контакты и адрес** — адрес, часы работы, телефон и email.
- ℹ️ **О нас** — краткая информация о компании.

---

## 🚀 Быстрый старт

### 1. Получение токена бота
1. Перейдите в Telegram и найдите бота [@BotFather](https://t.me/BotFather).
2. Напишите команду `/newbot` и следуйте подсказкам.
3. Скопируйте полученный **HTTP API token**.

### 2. Настройка проекта
1. Скопируйте файл конфигурации:
   ```bash
   cp .env.example .env
   ```
2. Откройте файл `.env` и вставьте ваш токен:
   ```env
   BOT_TOKEN=ваш_токен_от_botfather
   ```

### 3. Установка зависимостей
Установите необходимые библиотеки:
```bash
pip install -r requirements.txt
```

### 4. Запуск бота
Запустите скрипт:
```bash
python main.py
```

---

## 🛠 Структура проекта
- `main.py` — основной файл с логикой бота и обработчиками команд.
- `requirements.txt` — список необходимых библиотек (`aiogram`, `python-dotenv`).
- `.env` — файл с приватными настройками (токен бота).
- `.gitignore` — исключения для Git.
